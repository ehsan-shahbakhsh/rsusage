# Rust usage
Display RAM and CPU consumption

## Source
* [`main.rs`](https://github.com/ehsan-shahbakhsh/rsusage/blob/master/src/main.rs)

## Install dependencies
```shell
cargo install
```

## How to use
```shell
cargo run
```

## How to compile & release
```shell
cargo build --release
```
